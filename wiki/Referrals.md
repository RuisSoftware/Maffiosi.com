# 📨 Referral Systeem

Nodig vrienden uit voor Maffiosi.com en verdien fantastische beloningen! Het referral systeem is de makkelijkste manier om credits en bonussen te verdienen.

## Je referral link

Ga naar **Profiel → Referrals** om je persoonlijke referral link te vinden. Deze link ziet eruit als:

`https://maffiosi.com/ref/jouwgebruikersnaam`

Deel deze link met vrienden, op social media, of op forums. Elke speler die zich registreert via jouw link wordt jouw referral!

## Verificatie

Referrals moeten geverifieerd worden om te tellen. Een referral wordt geverifieerd wanneer de speler:
- Zijn/haar **e-mailadres** bevestigt
- Minimaal **rank 5** bereikt
- Minimaal **7 dagen actief** is geweest

Dit voorkomt misbruik en zorgt ervoor dat alleen echte, actieve spelers meetellen.

## Beloningen per referral

Elke geverifieerde referral levert je op:

| Aantal referrals | Beloning |
|-----------------|----------|
| 1e referral | 50 credits + €25.000 |
| 5 referrals | 100 credits + speciale badge |
| 10 referrals | 250 credits + zeldzaam item |
| 25 referrals | 500 credits + exclusieve titel |
| 50 referrals | 1.000 credits + uniek wapen |
| 100+ referrals | 5.000 credits + legendarisch item |

## Milestone Bonussen

Naast de individuele beloningen krijg je speciale **milestone bonussen** bij het bereiken van bepaalde aantallen. Deze bonussen stapelen — je ontvangt ze allemaal!

**Tip:** Deel je referral link in je Discord bio, op Maffiosi-gerelateerde forums, en vertel je vrienden over de game. Elke referral telt en brengt je dichter bij die legendarische rewards!
